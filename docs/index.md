# GoldenPipe

Golden Suite orchestrator — Check quality, fix issues, deduplicate records. One command.

GoldenPipe orchestrates the full pipeline with adaptive logic: it skips transformation if no quality issues are found, routes to privacy-preserving matching if sensitive fields are detected, and reports reasoning for every decision.

## Install

```bash
pip install goldenpipe
```

## Quick Start

```bash
goldenpipe run data.csv
```
