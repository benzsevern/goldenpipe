# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/benzsevern/goldenpipe/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

We will acknowledge your report within **48 hours** and provide a detailed response with next steps.
